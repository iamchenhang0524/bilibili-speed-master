# B站倍速大师 (Bilibili Speed Master) 🚀

这是一个专为 B 站（Bilibili）网页版设计的 Chrome 扩展程序，旨在将移动端极受好评的“长按倍速”体验完美迁移至桌面端。

## ✨ 核心功能

- **🔥 长按加速**：
  - 按住键盘 **`2`**：立即开启 **2.0x** 倍速播放。
  - 按住键盘 **`3`**：立即开启 **3.0x** 倍速播放。
- **⏪ 自动恢复**：松开按键后，视频将自动恢复到按下前的原始速度（无论你之前是 1.0x 还是 1.25x）。
- **🚫 智能防误触**：当你处于弹幕输入框、评论区或搜索框时，快捷键会自动失效，确保不影响正常打字。
- **🖥️ 视觉提示**：屏幕上方会有简洁的悬浮提示框，实时显示当前加速状态。

## 🛠️ 安装指南 (开发者模式)

由于本项目目前在 GitHub 开源，你可以通过以下步骤手动安装：

1. **下载项目**：点击页面右上角的 `Code` -> [Download ZIP](https://github.com/你的用户名/你的仓库名/archive/refs/heads/main.zip) 并解压到本地。
2. **进入插件管理**：打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
3. **开启开发者模式**：勾选页面右上角的 **“开发者模式 (Developer mode)”** 开关。
4. **加载插件**：点击左上角的 **“加载已解压的扩展程序 (Load unpacked)”**。
5. **选择文件夹**：在弹出的窗口中，选中你第一步解压得到的那个文件夹（包含 `manifest.json` 的文件夹）。
6. **完成**：刷新任何一个 B 站视频页面即可开始体验！
